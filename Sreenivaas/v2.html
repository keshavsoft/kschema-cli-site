kschema CLI

Scaffold backend projects instantly. Beginner friendly 🚀

Step 1: Run CLI

👉 Open your Command Prompt / Terminal
👉 Copy the below command and paste it in terminal
👉 Press Enter to run

npx @keshavsoft/kschema-cli

✅ After running the command, you will see options like:

kschema <init|test> [args]

📌 init → Create a new project
📌 test → Run test commands

Step 2: Create Project

👉 After running Step 1, your terminal will display options like:

kschema <init|test> [args]

👉 Now you need to choose init to create a new project
👉 Copy the below command and paste it in terminal
👉 Then press Enter

npx @keshavsoft/kschema-cli init kschema

🎉 This will create a new project folder named kschema
📁 Inside it, all required files and folders will be generated automatically

Step 3: Move to Project

👉 After Step 2, a new folder named kschema is created in your system
👉 But your terminal is still pointing to the old location (previous folder)

👉 Now you need to move inside the newly created project folder
👉 For that, use the cd (change directory) command

cd kschema

📌 cd means "change directory"
📁 This command moves your terminal into the kschema project folder

✅ Now your terminal is inside the project
👉 You can start working with project files like app.js, config.json

Step 4: Install Dependencies

👉 Now you are inside the project folder (kschema) after Step 3
👉 The project already has a package.json file

👉 This file contains all required libraries (dependencies) needed to run the project
👉 But they are not installed yet in your system

👉 To install all required packages, run the below command:

npm i

📦 This will download and install all dependencies into a folder called node_modules
⚙️ It reads from package.json and installs everything automatically

✅ After this step, your project is fully ready to run
🚀 You can now execute project files like app.js

Step 5: Open Project in VS Code

👉 After Step 4, all dependencies are installed and your project is ready
👉 Now it's time to open the project in a code editor

💻 Visual Studio Code (VS Code) is a popular code editor used by developers
👉 It helps you write, edit, and manage your project files easily

👉 Since you are already inside the project folder in terminal,
👉 you can open the project directly in VS Code using this command:

code .

📌 code . means "open current folder in VS Code"
📁 It will open the entire kschema project in the editor

✅ Now you can see files like app.js, config.json, Data folder
👉 You can start editing and running your project from here

⚠️ If the command doesn't work, make sure VS Code is installed and
👉 "Add to PATH" option is enabled during installation

Project Structure

kschema/
│── .vscode/        → VS Code settings
│── app.js          → main file
│── Config/         → config folder
│── config.json     → configuration
│── Data/           → data storage
│── package.json    → dependencies
│── read.js         → read logic
│── write.js        → write logic

How it Works

📥 read.js → Reads data

📤 write.js → Writes data

⚙️ config.json → Controls paths

📦 Data folder → Stores records
