kschema CLI

Scaffold projects instantly. Zero setup. Beginner friendly 🚀

Install

npm i -g @keshavsoft/kschema-cli

Quick Start

npx @keshavsoft/kschema-cli

After running CLI:

kschema <init|test> [args]

Choose one

npx @keshavsoft/kschema-cli init kschema
